# xyview

> vue2的UI插件,适合vue-cli创建的项目. 安装需要git客户端

> 配置scss

    npm i node-sass --save-dev
    npm i sass-loader --save-dev

> 安装

    npm i crossgit/xyview -save

> 卸载

    rm -rf node_modules/xyview

> 引入与使用,在main.js加入

    import * as xy from 'xyview' 
    Vue.use(xy.install)
 
 > 更多查看src目录中的项目实例.
